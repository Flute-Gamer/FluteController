This project aims to interpret flute notes into keyboards keys, to be presse while gaming.

The focus of this project is to play Sonic The Hedgehog, for the Sega Genesis, using my flute as a controller.

The Notes on the Flute play:


C 261 Hz -> Release Down key (Stop looking down)

D 293 Hz -> Hold Down key (Look Down and/or spin)

E 329 Hz -> Press Down key (Look Down and/or spin)



F 349 Hz -> Release Left key (Stop walking to the left)

G 391 Hz -> Hold Left key (Walk to the left)

G# 415Hz -> Hold Left key (Walk to the left)


A 440 Hz -> Hold Right key (Walk to the right)

B 493 Hz -> Release Right key (Stop walking right)

C 523 Hz -> Press Right key (Walk a little to the right)

G 783 Hz -> Press A key (Jump)

A 880 Hz -> Hold Up key (Look Up)

B 987 Hz -> Release Up key (Stop looking Up)

D 1174 Hz -> Press Enter key (Star/Pause)
